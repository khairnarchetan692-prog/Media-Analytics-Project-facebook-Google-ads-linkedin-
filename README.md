# Media Analytics Project

## Overview
End-to-end media analytics project analyzing user engagement, impressions, and campaign performance using Excel / SQL / Python / Power BI / Tableau.

## Dataset
- Sample dataset: `dataset/media_sample.csv`
- (Full dataset not included due to privacy/size)

## Tools
- Python (pandas, matplotlib)
- SQL
- Power BI / Tableau

## Analysis & Deliverables
- Data cleaning and ETL steps (`notebooks/`)
- SQL queries for summary metrics (`sql/`)
- Interactive dashboards (`dashboards/`)
- Visual screenshots (`screenshots/`)
- Final report (`report/Media_Analytics_Report.pdf`)

## How to run
1. Install dependencies: `pip install -r requirements.txt`
2. Open notebooks in `notebooks/`
3. Load sample data from `dataset/`
